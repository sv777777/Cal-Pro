## 使用说明
1. 复制链接
2. 粘贴到备忘录
3. 保存
4. 点击链接
5. 自动弹窗日历App
6. 订阅（可能会提示不安全链接，不影响，链接内容可以在浏览器直接访问到，是否安全都可以自己看到）

## 如果订阅失败，可重试多次，或卸载重装日历App，或更换网络重试
